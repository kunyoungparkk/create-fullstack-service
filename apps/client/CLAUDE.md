# 기술 스택

| 분류         | 기술            |
| ------------ | --------------- |
| Language     | TypeScript      |
| Framework    | React           |
| Build        | Vite            |
| Router       | TanStack Router |
| Remote State | TanStack Query  |
| HTTP         | axios           |
| Local State  | Zustand         |
| Styling      | Tailwind CSS    |
| i18n         | i18next         |
| Test         | Vitest          |

## 상태 관리

> 구현 패턴·쿼리 키 팩토리·커스텀 훅 작성법·무한 스크롤 등 상세는 **`state-management` 스킬**을 로드해 참고한다.
